<h1>MKS - CRUD FILMES</h1>
## Descrição

um api simples de um catalogo de filmes, onde um usuario autorizado pode criar, ver, editar e deletar um filme

## Instalação
necessario ter docker instalado na maquina 
```bash
$ npm install

# container docker
$ docker compose up -d 
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
